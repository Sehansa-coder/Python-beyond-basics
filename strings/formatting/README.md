# Python string formatting
This folder contains examples of **string fomatting techniques in Python**, ranging from legacy methods to modern best practices.

String formatting is commonly used for:
 -Displaying output
 -Formatting numbers and dates
 -Debugging
 -Creating clean, readable messeges

-**Examples with explanations are included**.
---

## Files included

-**percent_formatting.py**
Old style (%) string formatting (legacy syntax)

-**using_format_function.py**
Formatting using the '.format()' method.

-**using_f_string.py**
Modern f-string formatting(recommended)

-**number formatting**
Formatting numbers, decimals, and thousand seperators.

-**formatting_dates_and_time.py**
Formatting dates and times for real world use

-**debug_formatting.py**
Debug formatting using f-string('{variable=}').

-**multiline_f_string.py**
Clean multi-line output using f-strings.

-**escape_characters.py**
Using escape characters ('\n','\t','\\',qoutes).

---

# Note
For modern Python development, **f-strings are the preferred and recommended approach** due to their readability and simplicity.
